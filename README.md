# 📦Sistema Gestion De Tiendas
Este es un proyecto que busca ayudar a las personas las cuales poseen un negocio en el cual se gestionen ventas, productos y diferentes reportes o estadisticas.

# 🏗️ Estructura del proyecto
* **Backend (`/backend`)**: Construido en Java y Spring boot, siguiendo los principios de la Arquitectura Hexagonal. Lo que permite que la logica de negocio este lo mas aislada posible de la infraestructura.

* **Frontend (`/frontend`)**: Contruido en React con TypeScript, ademas de utilizar bootstrap para un mejor diseño.

* **Documentación (`/docs`)**: Aqui se encontraran tanto imagenes, archivos, documentos que ayudan al entendimiento de la estructura y el funcionamiento del sistema.

Cada proyecto por separado tiene un readme donde se explica en detalle el proyecto


# 🧠 Motivation & Learning
Este proyecto surge gracias a querer poner en práctica todos aquellos conocimientos de software, como lenguajes de programación (tanto de front como de backend), ademas de patrones y arquitecturas, ciertas convenciones y/o buenas practicas que fui adquiriendo en lo que voy de carrera.

Me interezo mucho ver como los distintos patrones y arquitecturas permiten que el codigo sea altamente escalable y mantenible. Lo cual ayuda a que los sistemas sean más robuztos y funcionen de mejor manera sin tener que estar modificando tanto el software.

Por ello me decante a elegir una **Arquitectura Hexagonal** en el backend, ya que esta nos permite centrarnos en lo que mas importa que es el negocio. Abstrayéndose de lo externo, como lo serian las base de datos, la interfaz de usuario, los distintos metodos de comunicacion entre front y back, entre otros.

Si bien he hecho varios trabajos practicos donde se desarrollaba software aplicando estas arquitecturas o patrones y lenguajes de programacion, sentia que necesitaba hacerlo por mi cuenta rebuscando informacion de entre mis apuntes, libros de las catedras, internet, inteligencia artificial y buscar de hacer lo mejor posible con lo que tenia a mano.

Espero este proyecto me pueda ayudar a mi a mejorar como estudiante y profesional, ademas de a mis compañeros para entender un poco mejor de lo que van viendo en la carrera aplicado a un proyecto "real".
